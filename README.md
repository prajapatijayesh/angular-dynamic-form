# Angular Dynamic Form

Dynamically Building Forms

## Running the Sample
<ol>
  <li>Clone the Git repository</li>
  <pre><code>
  git@github.com:prajapatijayesh/angular-dynamic-form.git
  </code></pre>
  <li>Install the npm dependencies</li>
  <pre><code>
  npm install
  </code></pre>
  <li>Run the application</li>
  <pre><code>
  ng serve
  </code></pre>
</ol>
